# hubot-gobot

GoCD integration bot

See [`src/gobot.coffee`](src/gobot.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-gobot --save`

Then add **hubot-gobot** to your `external-scripts.json`:

```json
[
  "hubot-gobot"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
